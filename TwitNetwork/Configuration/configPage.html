TWiT.tv API Configuration

TWiT Developer Program

In order to use this plugin, you will need to set up your own access account for the TWiT API at 3scale. Please see above link for more information, they tend to be reasonably quick in allowing access. There is currently no charge to use the TWiT API, although it is rate limited to 5 requests per minute.
